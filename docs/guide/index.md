# Documentation

Welcome to the LokulMem docs. This section is structured so you can go from install to production guardrails quickly.

## What LokulMem provides

- A browser-native memory layer for conversational AI apps.
- Deterministic extraction and write policy with optional fallback extraction path.
- Canonicalization, contradiction handling, and supersession lineage.
- Eval gates and CI checks to protect quality over time.
