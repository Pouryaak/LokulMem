<objective>
Implement token-aware dynamic K selection that accounts for LLM context window limits.

Purpose: Enable LokulMem to automatically calculate how many memories can be injected based on the actual context window of the LLM being used (e.g., 4096 for GPT-3.5, 8192 for GPT-4), system prompt size, and user message length. This eliminates the need for users to manually calculate and pass maxTokens parameter.

Output: Context-aware token budget calculation in getInjectionPreview() with configurable contextWindow and systemPromptTokens in LokulMemConfig.
</objective>

    3. Update getInjectionPreview() signature and implementation:
       ```typescript
       async getInjectionPreview(
         query: string,
         options?: {
           maxTokens?: number; // Override automatic calculation
           systemPromptTokens?: number; // Override config
           userMessage?: string; // For accurate token estimation
         }
       ): Promise<{
         memories: MemoryDTO[];
         estimatedTokens: number;
         remainingTokens: number; // New: tokens left after injection
       }> {
         const {
           maxTokens: maxTokensOverride,
           systemPromptTokens: systemPromptOverride,
           userMessage
         } = options ?? {};

         // Calculate available token budget
         const systemPromptTokens = systemPromptOverride ?? this.systemPromptTokens;
         const userMessageTokens = userMessage ? this.estimateTokens(userMessage) : this.estimateTokens(query);

         // Calculate remaining tokens after system prompt and user message
         const remainingTokens = this.contextWindow - systemPromptTokens - userMessageTokens;

         // Allow manual override via maxTokens
         const maxTokens = maxTokensOverride ?? Math.max(0, remainingTokens);

         const results = await this.semanticSearch(query, { k: 50 });

         // Dynamic K based on token budget
         const limitedMemories: MemoryDTO[] = [];
         let currentTokens = 0;

         for (const memory of results) {
           const memTokens = this.estimateTokens(memory.content);
           if (currentTokens + memTokens > maxTokens) break;
           limitedMemories.push(memory);
           currentTokens += memTokens;
         }

         return {
           memories: limitedMemories,
           estimatedTokens: currentTokens,
           remainingTokens: remainingTokens - currentTokens
         };
       }
       ```
