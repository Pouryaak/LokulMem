    3. calculateDecay(memory: MemoryInternal, now: number): DecayResult
       - Get oldStrength from memory.currentStrength
       - Determine lambda: 0 if pinned, otherwise getLambdaForTypes(memory.types)
       - Calculate timestamp: memory.lastAccessedAt || memory.createdAt
       - Calculate ageHours: (now - timestamp) / (1000 * 60 * 60)
       - Compute decayFactor: Math.exp(-lambda * ageHours)
       - Compute newStrength: memory.baseStrength * decayFactor
       - Determine isFaded: newStrength < config.fadedThreshold && !memory.pinned
       - Return DecayResult with all fields

    4. calculateDecayBatch(memories: MemoryInternal[], now: number): DecayResult[]
       - Map over memories, call calculateDecay for each
       - Return array of DecayResult

    5. Private getLambdaForTypes(types: MemoryType[]): number
       - Start with DEFAULT_LAMBDA
       - Iterate through types, find minimum lambda from config.lambdaByCategory
       - Return minimum lambda (use smallest for memories with multiple types)
       - Use DEFAULT_LAMBDA if no types or no matches

    6. Private validateLambda(lambda: number): void
       - Throw error if lambda < 0
       - Call this in constructor for all lambda values in config

       - Return early if memory.baseStrength >= config.maxBaseStrength
       - Get primary category: memory.types[0] || 'preference'
       - Get reinforcement amount: config.reinforcementByCategory[category] || 0.3
       - Return early if baseStrength + reinforcement > config.maxBaseStrength
       - Add pendingReinforcement: { memoryId, category, timestamp: Date.now() }
       - Call scheduleDebouncedWrite()

    4. Private scheduleDebouncedWrite(): void
       - Return early if debounceTimer already set
       - Set timer for config.debounceWindowMs
       - On timeout, call flushPendingReinforcements() and set timer to null

    5. Private async flushPendingReinforcements(): Promise<void>

       - Convert Map values to array, clear Map
       - Fetch all memories by ID using repository.getById
       - Filter out null results
       - For each memory, calculate newBaseStrength with cap:

       - Update memory: baseStrength, lastAccessedAt: Date.now(), mentionCount + 1
       - Call repository.bulkUpdateStrengths(updates)
       - Log count of reinforcements applied

    6. async forceFlush(): Promise<void>
       - Clear debounceTimer if set
       - Call flushPendingReinforcements()

    7. getPendingCount(): number
       - Return pendingReinforcements.size

    async bulkUpdateStrengths(memories: MemoryInternal[]): Promise<void> {
      if (memories.length === 0) {

        const row = memoryToDb(memory);
        // Only update strength-related fields
        return {

          // Keep all fields but we'll only put these in the update call

      });
