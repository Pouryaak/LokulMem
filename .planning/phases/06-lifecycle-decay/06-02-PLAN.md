  <files>src/lifecycle/MaintenanceSweep.ts</files>
  <action>
    Create src/lifecycle/MaintenanceSweep.ts with MaintenanceSweep class:

    1. Class properties:
       - Private config: MaintenanceConfig
       - Private decayCalculator: DecayCalculator
       - Private reinforcementTracker: ReinforcementTracker
       - Private repository: MemoryRepository
       - Private eventEmitter: LifecycleEventEmitter
       - Private periodicTimer: ReturnType<typeof setInterval> | null
       - Private isSweepRunning: boolean = false

    2. Constructor: Accept all dependencies

    3. async runSweep(): Promise<SweepResult>
       - Set isSweepRunning = true, return if already true
       - Call onProgress('decay', 0)
       - await reinforcementTracker.forceFlush()
       - Get all memories via repository.getAll()
       - Calculate decay results: decayCalculator.calculateDecayBatch(memories, now)
       - Call onProgress('decay', 50)
       - Separate faded memories and build updates array:
         - Set memory.currentStrength = result.newStrength
         - If result.isFaded && memory.status === 'active': set status='faded', fadedAt=now, add to fadedMemories
         - Add memory to updates array
       - await repository.bulkUpdateCurrentStrengths(updates)
       - Call onProgress('decay', 75)
       - Emit fade events: for each faded memory, await eventEmitter.emitMemoryFaded()
       - Call onProgress('decay', 90)
       - deletedCount = await deleteOldFadedMemories(now)
       - Call onProgress('decay', 100)
       - Set isSweepRunning = false
       - Return { decayedCount: updates.length, fadedCount: fadedMemories.length, deletedCount }

    4. Private async deleteOldFadedMemories(now: number): Promise<number>
       - Calculate cutoffTime: now - (30 * 24 * 60 * 60 * 1000)
       - Get faded memories: repository.findByStatus('faded')
       - Filter where fadedAt !== null && fadedAt < cutoffTime
       - Return 0 if empty
       - Extract IDs, call repository.bulkDelete(idsToDelete)
       - Emit delete events: for each memory, await eventEmitter.emitMemoryDeleted(memory.id)
       - Log count deleted
       - Return count

    5. startPeriodicSweeps(): void
       - Return if periodicTimer already set
       - Set timer with setInterval(config.sweepIntervalMs)
       - On interval, await runSweep() with try-catch
       - Log started message

    6. stopPeriodicSweeps(): void
       - Clear periodicTimer if set
       - Set to null
       - Log stopped message

    7. async shutdown(): Promise<void>
       - Call stopPeriodicSweeps()
       - await reinforcementTracker.forceFlush()
